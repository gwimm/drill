# drill

a lone drill

works for:
- bandcamp

## Installation

```
git clone github.com/gwimm/drill.git
cd drill
cargo build
```

## Usage

```
 ~ > drill -h
USAGE:
    drill [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -u, --url <urls>...    set url
```
